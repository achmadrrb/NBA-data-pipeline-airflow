# NBA Data Scraping Pipeline using Beautiful Soup, Airflow, and BigQuery


## Requirement

- uv
  - [Installation Guide](https://docs.astral.sh/uv/getting-started/installation/)
- Python v3.12.\*

## Setup Environment

```bash
uv sync
```

### 2. Install Pre-Commit

```bash
uv run pre-commit install
```
